# MintDownload
项目开始，使用flutter。


本项目是一个安卓应用程序。

虽然有意愿，但暂时不会考虑制作IOS版本，因为没有苹果设备。


本项目实际意义不大，目的主要是练手。



### 目前的开发目标
一个轻量级的下载程序。功能如下：

* 输入http链接，能下载对应链接的文件到手机里。如果不是文件链接，将会发出提示。

* 显示下载速度，以及是否正常的提示。

* 支持断点续传。

* 下载目录可更改。

* 当下载失败，显示相应的提示信息。
